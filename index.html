About the Project

About SiteSpy

SiteSpy is a mobile-first construction project records and cost estimation app built with Expo React Native, Firebase, and Firestore. It helps construction teams capture measurements, estimate material costs, organise site records, and review project history on any device — even in low-connectivity field conditions.

My Contribution

My Contribution

I took ownership of documenting the Firestore data model that SiteSpy relies on. This means writing clear, accurate documentation about how projects, estimates, and user records are structured in the database. I also set up and reviewed the Firestore indexes (firestore.indexes.json) that ensure queries run efficiently, and produced the Firebase Setup Guide so future developers can configure the backend without guessing.

Reflection

My Reflection

Documentation work looks unglamorous but it is where knowledge lives. I found that writing precise data model descriptions forced me to deeply understand how each Firestore collection relates to the others. The index configuration work taught me that bad indexes cause query timeouts that look like bugs — clear documentation prevents that.
